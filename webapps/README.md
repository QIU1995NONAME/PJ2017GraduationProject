# Web Apps

## 常用命令

环境相关
```bash
npm install -g yarn                # 安装 Yarn
npm install -g @vue/cli            # 安装 Vue CLI
```

创建工程
```bash
vue      create   "project-name"   # 创建 Vue 工程
npm init vite-app "project-name"   # 创建 Vite 工程
```

在工程目录里面添加依赖
```bash

vue  add vuex                      # 添加 Vue 插件
vue  add router
vue  add bootstrap

yarn add echarts                   # 添加依赖
```
